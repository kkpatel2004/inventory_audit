### Company Inventory Audit – Overview

## Overview

The **Company Inventory Audit System** is a software application designed to help organizations efficiently monitor, verify, and manage their inventory records. The system streamlines the auditing process by comparing physical inventory with recorded stock, identifying discrepancies, and generating detailed audit reports.

This project aims to improve inventory accuracy, reduce manual errors, and provide better visibility into stock movements. It enables businesses to maintain accurate inventory records while supporting informed decision-making and regulatory compliance.

### Key Features

* **Inventory Management** – Add, update, delete, and view inventory items.
* **Audit Tracking** – Record and manage inventory audit activities.
* **Stock Verification** – Compare actual stock with recorded inventory.
* **Discrepancy Detection** – Identify missing, excess, or mismatched inventory items.
* **Report Generation** – Generate audit reports for analysis and record-keeping.
* **User Authentication** – Secure login for administrators and authorized users.
* **Dashboard** – Display inventory statistics and audit summaries.

### Objectives

* Automate the inventory auditing process.
* Improve inventory accuracy and accountability.
* Minimize manual data entry and human errors.
* Generate comprehensive audit reports.
* Enhance operational efficiency through real-time inventory monitoring.

### Technologies Used

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Python (Flask)
* **Database:** MySQL
* **Version Control:** Git & GitHub

### Benefits

* Faster and more efficient inventory audits.
* Improved data accuracy and transparency.
* Easy tracking of inventory discrepancies.
* Better decision-making through analytical reports.
* Secure and centralized inventory management.
